windows下node和npm升级

2017-08-24

在使用Vue框架时，手动安装Vue调试工具的环境要求Node 6+ &NPM 3+,故要升级本地Node和NPM。注意，在node目录下操作是报错的，后转到管理员目录下操作才行。

1、查看当前node和npm版本

  • xxx@xxx MINGW32 /c/users/Administrator
  • $ npm -v
  • 2.15.1
  • xxx@xxx MINGW32 /c/users/Administrator
  • $ node -v
  • 4.4.3

2、升级npm

  • xxx@xxx MINGW32 /c/users/Administrator
  • $ npm install npm -g
  • C:\Users\chaowang211311\AppData\Roaming\npm\npx -> C:\Users\chaowang211311\AppData\Roaming\npm\node_modules\npm\bin\npx-cli.js
  • C:\Users\chaowang211311\AppData\Roaming\npm\npm -> C:\Users\chaowang211311\AppData\Roaming\npm\node_modules\npm\bin\npm-cli.js
  • npm@5.4.1 C:\Users\chaowang211311\AppData\Roaming\npm\node_modules\npm

3、升级node

#安装全局n模块
  • xxx@xxx MINGW32 /c/users/Administrator
  • $ npm install -g n
  • npm ERR! Buffer.alloc is not a function
  • npm ERR! A complete log of this run can be found in:
  • npm ERR! C:\Users\chaowang211311\AppData\Roaming\npm-cache\_logs\2017-09-08T08_48_11_934Z-debug.log
  • #报错,so,执行以下命令
  • xxx@xxx MINGW32 /c/users/Administrator
  • $ npm install -g n --force

#升级最新稳定版本
  • //升级到最新稳定版本
  • n stable
  • //升级到最新版本
  • n latest
  • //升级到指定版本
  • n 5.1.0
