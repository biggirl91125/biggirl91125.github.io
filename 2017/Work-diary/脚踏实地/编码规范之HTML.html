编码规范之HTML

2017-07-07

磨刀不误砍柴工，我们需要静下心来扎实一下基本功。

1、语法

  • 用两个空格替代制表符(Tab)，保证适用所有环境
  • 属性定义使用双引号，切勿使用单引号
  • 自闭合(Self-closing)元素无需在尾部添加斜线

2、HTML5 DOCTYPE

为每个HTML页的第一行添加标准模式(Standard Mode)的声明,确保浏览器解析一致。

  1. <!DOCTYPE html>
  2. <html>
  3. <head></head>
  4. <body></body>
  5. </html>

3、语言属性

强烈建议为html根元素指定lang属性,从而为文档设置正确的语言,有助于指导浏览器进行正确行为，如语音合成工具确定其所采用的发音，翻译工具确定其翻译规则。

  1. <!DOCTYPE html>
  2. <html>
  3. <!-- -->
  4. </html>
